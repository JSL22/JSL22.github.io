Quick Start

Create a new post

（0）清屏  ： clear  
（1）初始化Git仓库输入 ：   git init

（2）设置用户和邮箱 ： git config --global user.name "jishaling"
                     git config --global user.email "13137755917@163.com"
（3）添加代码   ： git add   文件路径     路径写为“/”
               : git add ./        添加所有修改的文件
    提交到仓库  : git commit -m "介绍文件信息"
（4）查看当前状态： git status            （检查文件是否添加或提交）
 (5) 直接提交到仓库 ： git commit --all -m "介绍文件信息"
（6）查看操作信息  ： git log         git log --oneline
 (7) 重置到修改前的文件 ： git reset --hard Head~0     0可以修改  
                         0最后修改的文件 ；1 重置到倒数第一次提交的文件 
                         2 重置到倒数第二次提交的文件 .... 

                        git reflog 
                        git reset --hard 提交版本号 
 (8) 创建分支  ： git branch 分支名
     查看分支  ： git branch        * 为当前分支
     切换分支 ： git checkout  分支名
     合并分支  ： git merge  分支名
     删除分支  ： git branch -d 分支名

 (9) 传送代码  : git push  地址  分支名
     获取代码  : git pull  地址  分支名       （初始化一个仓库？）
                git clone  地址  分支名         （多次执行会覆盖）
        git remote add  origin 地址  （将地址命名为origin，在当前的）
        git push origin -u master  (第一次，形成关联)
        git push (第二次)
